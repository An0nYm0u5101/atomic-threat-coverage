detection:
    selection1:
        CommandLine:

### powershell
    
```
Get-WinEvent | where {(($_.message -match "CommandLine.*.*netsh.*") -and ($_.message -match "CommandLine.*.*firewall add.*")) } | select TimeCreated,Id,RecordId,ProcessId,MachineName,Message
```
